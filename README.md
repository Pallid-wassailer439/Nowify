## 🎯 What Nowify Does

Nowify shows your current Spotify track as a live overlay for OBS. It fits into streams, recordings, and panels with a clean look. You can change the design, switch presets, and use it with Twitch tools.

It is made for streamers who want a simple way to show music on screen without extra setup.

## 🧩 Add Nowify to OBS

Once Nowify is running, add it to OBS as a browser source:

1. Open OBS.
2. Go to your scene list and choose the scene you want.
3. In the Sources box, click the plus button.
4. Choose Browser.
5. Give it a name like Nowify.
6. Paste the overlay link from Nowify into the URL field.
7. Set the width and height to match your layout.
8. Click OK.
9. Move and resize the overlay in the preview window.

If the overlay has a transparent background, it will sit on top of your scene and blend with your layout.

## 🛠️ Basic Setup Tips

Use these tips if the overlay does not look right the first time:

- Make the browser source the right size

- Check that Spotify is playing music

- Place the overlay where it does not cover key content

If you use multiple scenes, copy the same browser source settings into each one for a matching look.

## 🧭 Troubleshooting

If Nowify does not show the song:

- Confirm Spotify is playing
- Refresh the browser source in OBS
- Restart Nowify
- Restart OBS
- Check that the overlay link is still correct

If the overlay looks too small or too large:

- Change the browser source width and height
- Adjust the scale in OBS
- Pick a preset that matches your scene size

If the text is hard to read:

- Use a stronger text color
- Add a background
- Increase font size
- Move the overlay away from bright parts of your scene

If the cover art does not appear:

- Check the preset settings
- Refresh the source
- Confirm Spotify has album art for the track
